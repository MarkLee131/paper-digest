# Paper Digest

A daily reading log for software-engineering, programming-languages, and security research.

Each entry triages the day's new arXiv / venue announcements with a two-stage rubric — fast title/abstract screening, then deep-read with meta-cognition (problem formulation, solution-space reduction, reusable pattern). Outstanding papers get a full overview, formal characterisation, system-overview diagram, and workflow walkthrough; lighter passes go to *Keep* and *Borderline* entries.

The full archive is browsable at **[marklee131.github.io/paper-digest](https://marklee131.github.io/paper-digest/)**.

Built with [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) and deployed via GitHub Pages.
